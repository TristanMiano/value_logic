General relativity and quantum effects are also tiny.

But drag may be large enough to matter. So a vacuum Newtonian model may fail not because of relativity, but because of air resistance.

That is exactly what a record-derived embedding should discover: the dominant inadequacy axis for classroom projectiles is likely drag, not relativity.

For GPS, by contrast, the tolerance is much stricter. Even tiny relativistic corrections matter. So the Newtonian tile shrinks until it no longer covers the GPS domain.

The same physical world has shifted domains because the tolerance changed.

[
\text{classroom projectile}
\neq
\text{GPS timing problem}
]

even if both involve macroscopic objects moving in Earth‚Äôs gravitational field.
The deep network version

The clean neural architecture is:

[
x=\varphi(y)
]

where (x) is preferably built from log dimensionless groups when physics gives them.

Then each correction detector computes:

[
q_c(x)=\langle e_c,x\rangle+k_c.
]

This estimates the log-size of omitted correction (c).

Each model has a set of omitted corrections (C(M)). Its predicted log-loss is:

[
\log \widehat L(M,y)
\approx
\max_{c\in C(M)} q_c(x).
]

Its adequacy margin is:

\log\epsilon-
\max_{c\in C(M)}q_c(x).
]

Then:

[
\ell(M,y,\epsilon)=\operatorname{ReLU}(s(M,y,\epsilon)).
]

A deeper network can learn the residual map:

[
\psi(\log\pi_1,\dots,\log\pi_k)
]

when the boundary is curved or when the obvious dimensionless groups do not fully capture the regime.

So depth is not replacing the physics. Depth handles the leftover structure after symmetry, units, and perturbation theory have done their work.
General recipe

The practical recipe is:

    Assemble a bank of records (y_j) and observed outcomes (o_j).

    Run each model (M_i) on each record.

    Store the loss tensor:

[
L_{ij}=L(M_i,y_j).
]

    Learn embeddings by fitting:

c_i+r_j-\langle m_i,\varphi(y_j)\rangle.
]

    Gate by tolerance:

\log\epsilon-\log\widehat L_i(y).
]

    License models with:

[
\operatorname{ReLU}(s_i)>0.
]

    Abstain if no model has positive margin.

    Monitor residuals. If errors are structured, add a new axis, split a domain, or introduce a new model/correction.

This turns conceptual innovation into geometry. A new theory or correction appears when the old embedding leaves structured residuals that cannot be compressed into the current coordinates.
