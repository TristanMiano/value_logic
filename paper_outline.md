## 1. Publication thesis and claim hierarchy

### 1.1 One-sentence thesis

A bounded agent can represent present, revisable permission to rely on a model
without asserting finality: a compact profile-indexed calculus makes domain,
task, evidence, fallback, comparison, and abstention explicit; an
architecture-neutral factorization admits an exact finite ReLU reference
witness for suitable numerical statistics while leaving evidence and decoding
external; and the frozen experiment shows that reusable numerical information,
calibrated caution, and operational coverage can separate sharply.

### 1.5 Optional policy/value case study

The policy/value bridge is an independent, bounded application and research
motivation. It cannot become a second main theorem spine.

1. **Finite representation existence retained.** For finite action alphabet
   `\mathcal A`, any deterministic policy `pi:X -> \mathcal A` has a lossless real-valued
   encoding after fixing an injective action code; equivalently, its chosen
   action can be represented by a finite action-score vector with a fixed
   tie-breaking decoder. Restricting the codomain to the encoder image yields
   inverse maps. This is a representational existence proposition.
2. **Separate questions remain separate.** Standard pointwise `V^pi` and
   `Q^pi` require a declared environment, reward/return, sufficiently Markov
   state, horizon or discount, and perspective. Occupancy additionally depends
   on an initial or evaluation distribution. Greedy use of `Q^pi` can improve a
   suboptimal policy rather than reconstruct it. Normalized full state--action
   occupancy can recover a Markov policy only on positively occupied states;
   state occupancy alone cannot recover its action choice, and neither object
   is automatically utility. The existence proposition supplies
   neither a unique or natural scalar geometry, reward identification,
   standard-return invertibility, practical learnability, nor mechanistic
   interpretation.
3. **No claim about true utility.** The project does not investigate whether
   an arbitrary policy has a true utility function. It makes no claim that a
   true utility exists, that a surrogate recovers it, or that unlimited compute
   would identify a uniquely truest recoverable approximation.
4. **Positive practical target.** The proposed object is an approximate,
   environment-relative semantic surrogate of a bounded black-box policy.
   Value was the first semantic object the author saw how to reconstruct
   directly enough to motivate the pragmatic logic. This is transparent
   research history and interpretive motivation; it makes no claim that value
   is metaphysically prior to belief or that belief extraction is impossible.
5. **Evidence remains graded.** Behavioral fidelity, value fidelity,
   outcome/task-quotient information, domain validity, representational
   alignment, causal faithfulness, and human inspectability are distinct. Task
   23 specifies tests but supplies no positive causal or human evidence.
6. **Operational reconstruction is available at a conditional scope.** Task
   22B proves the oracle distribution-scoped bound
   `D_mu<=mu(e>rho)+mu(g<=2rho)` and gives accepted event-mass, pointwise, and
   held-out IID certificate routes. Public prose may call the result certified
   only under one of those evidence premises. It must distinguish raw `2 rho`
   argmax stability from the generic `4 rho` sufficient condition for a
   conservative decoder to recover and avoid abstention, and it must state the
   full decision harness. This remains an optional integration result.
7. **The inward semantic program is future work.** The author's motivating
   hope is that a licensed value-like output can anchor later recovery of
   semantic structure deeper in a network. Work inside an independently trained
   surrogate describes that surrogate; a claim about the original policy also
   requires an explicit hidden-state alignment/readout link and policy-side
   interventions. Those are separate empirical gates.

### 1.6 Compact standard-mathematics motivation from recursive judgment

The optional case-study section may state one result. Let `R` be a report, `Y`
a held-out outcome, and `N` pre-outcome nuisance information. If a valid
evaluation certificate lower-bounds the population log-loss improvement over
the true Bayes predictor based on `N` by `delta>0` nats, then

```text
I(R;Y | N) >= delta.
```

If all report--outcome dependence is mediated by a latent task `Z` conditional
on `N`, let `Z_bar` merge task labels with identical conditional outcome laws.
Then data processing gives

```text
I(R;Z_bar | N) >= I(R;Y | N) >= delta.
```

This is standard proper-score/KL/data-processing mathematics applied to the
project. It is not advertised as a new information-theory contribution, and
`Z_bar` is only the outcome-identifiable task quotient. Put this countermodel
next to it: if `R=Y` through direct leakage while `Z` is constant, the report
predicts perfectly but contains no task information; the mediation condition
fails. The appendix adds the non-Bayes comparator, omitted nuisance,
duplicated-label, instability, and recursive-copying cases. In particular, a
copied prior report can agree perfectly while adding zero information
conditional on that prior report.

If an empirical application uses a merely `N`-measurable comparator `b`, the
paper states the practical correction
`I(R;Y|N) >= delta_b-Regret_N(b)` for the population risk gap `delta_b`.
Empirically, a joint certificate with population-gap lower bound `L_delta` and
comparator-regret upper bound `U_regret` gives
`I(R;Y|N)>=L_delta-U_regret`. “Best declared baseline” is not sufficient
wording.

## 2. Frozen reader-facing formulations

### 2.1 Plain-language answer: “Why this value-logic design, and why hope it fits neural networks?”

Physics often advances by limiting the reach of an older theory rather than erasing every use of it. A model can lose a claim to universal correctness while remaining dependable on a restricted domain, for a stated purpose and error tolerance. The proposed value logic records a bounded agent's current permission to rely on such a model. That permission is indexed by the model, domain, task loss, evidence, fallback, constraints, provenance, and finite library searched. The project remains neutral about whether the present theory is finally true.

A tolerance gets an operational meaning from what the agent would otherwise do. If a fallback has expected loss `.35` and a candidate must improve on it by `.05`, the fallback-derived threshold is `epsilon_B=.30`. A separate adequacy rule may require loss at most `.20`, while a constraint requires latency at most `50 ms`. In the running synthetic succession, an older local plan and a broader successor initially satisfy all three requirements on an overlap. Both can be licensed even if a router selects one. An uncovered region invokes the fallback. When the older plan's certificate expires, reliance is withheld because evidence has lapsed. When accepted evidence later places its loss above the boundary, reliance is refused. Tightening adequacy from `.20` to `.16` makes the older interval straddle the boundary while the successor is supported exactly at equality, without retraining. A later specialist fills the earlier gap and becomes preferable on a certified finite overlap while the older plans remain recorded.

This structure motivates a hybrid neural interface: a learned module proposes reusable quantities such as loss statistics and uncertainty regions, while an exact external layer checks evidence, well-formedness, inclusive boundaries, required atoms, masking, and fallback. ReLU is one useful reference because finite ReLU networks compute piecewise-linear statistics and expose positive margin surplus. The semantics still come from the named quantity and accepted evidence; supported equality, an unresolved interval, and missing evidence can all have zero ReLU surplus, so exact state and provenance must remain beside it. One frozen synthetic implementation received the following asymmetric verdict: no-retraining tolerance generalization was supported; boundary superiority and in-regime noninferiority were refuted at their registered margins; marginal proposal coverage was supported; and usable coverage was poor, with `0.9962` target-weighted fallback mass. **Retaining a reusable numerical statistic helped when the decision threshold changed. But wrapping that statistic in a conservative uncertainty-and-decoding pipeline often turned informative predictions into abstentions. Representational information, calibrated caution, and operational usefulness are separate achievements.** Conservative dead-band geometry is a plausible observed mechanism, but the experiment did not identify a causal decomposition or establish that ReLU is uniquely suitable.

An optional policy/value bridge asks whether an environment-relative value surrogate can give a useful semantic view of a black-box policy. The finite existence construction shows that a lossless value-like code is possible. Standard return semantics, identification, and learnability require additional assumptions. The project does not investigate whether true utility exists or claim to recover it. Behavioral, value, outcome/task-quotient, domain, representational, causal, and human evidence remain distinct grades.

### 2.2 Abstract-level version (100--150 words)

Scientific models can be superseded while remaining useful on restricted domains. We formalize this situation as a finite-stage, evidence-relative license indexed by model, domain, task loss, fallback, tolerance, profile, and provenance. An architecture-neutral factorization lets learned modules propose reusable numerical statistics while exact checks preserve well-formedness, atom states, masking, and fallback; finite ReLU networks provide one reference realization. Retaining a reusable numerical statistic helped when the decision threshold changed, but the conservative uncertainty-and-decoding pipeline often converted informative predictions into abstentions. In the frozen synthetic experiment, no-retraining tolerance transfer and marginal proposal coverage were supported; boundary superiority and in-regime noninferiority were refuted at their registered margins; usable coverage was poor. Formal results characterize profile refinement, open-ended stability, update locality, robust representation, and boundary obstructions. The project makes no claim of architectural optimality, final truth, or true-utility recovery.

### 2.3 Running succession example: use this one everywhere

The public example uses descriptive names to prevent notation collisions:

```text
M_old:  older plan, useful on a local domain
M_succ: broader successor with an overlapping domain
M_new:  later specialist, unavailable before stage 3
B:      declared fallback
```

It is “Newtonian-like” only in the abstract succession pattern. The numbers are
synthetic and make no claim about the measured accuracy of Newtonian,
relativistic, quantum, Lagrangian, or Hamiltonian models.

The fallback comes before the neural example. Let `J(e)` be smaller-is-better
task loss. The fallback has `J(B)=.35`, and a candidate must improve by
`Delta=.05`, hence

```text
epsilon_B = J(B)-Delta = .30.
```

The request also imposes independent adequacy and latency requirements:

```text
Adeq:    J(e) <= .20
Improve: J(e) <= .30
Latency: T(e) <= 50 ms.
```

At stage 0 on an overlap,

```text
M_old:  U_J=[.14,.18], U_T=[43,47]
M_succ: U_J=[.11,.16], U_T=[45,49].
```

Both are `Granted`; this permits simultaneous reliance and does not force a tie
or a particular selector. Elsewhere an intentional gap has no active plan and
therefore invokes `B`. A wrong-unit request is `Undefined`, while a missing
latency record makes a well-formed request `Withheld`.

At stage 1, expiry of `M_old`'s loss certificate makes the relevant atom open
and the request `Withheld`: the warrant lapsed. An update outside a request's
typed footprint leaves its complete diagnostic unchanged. At stage 2, new
two-sided evidence `U_J(M_old)=[.23,.25]` lies beyond `.20`, so the request is
`Refused`: this is a rebuttal. These are distinct update paths.

For tolerance transfer, re-evaluate the original intervals after changing the
adequacy boundary from `.20` to `.16`. `M_old` now straddles the boundary and is
open; `M_succ` is supported at inclusive equality. Neither learner is
retrained. This is the bridge from a reusable statistic to changed decisions.

At stage 3, `M_new` covers the earlier gap. Checked paired-difference
certificates have strictly negative upper endpoints for both loss and latency
against the displayed `M_old/M_succ` candidates, so `M_new` dominates that
exact finite evaluated set on the declared overlap. The old plans remain in
the atlas and can retain basic licenses on other scopes. Finite comparative
preference changes selection; it does not erase history or establish global
optimality against unexamined future plans.

The ReLU boundary witness then reuses the same example. For an accepted region
`U_J=[l,u]`, define signed margins

```text
m_support = epsilon-u,
m_refute  = l-epsilon.
```

An exact decoder includes equality on the supported side and uses strict
inequality for refutation. `ReLU(m_support)>0` is strict positive surplus for
the named atom when the envelope and evidence are accepted. It is not the full
license. At supported equality, an open crossing, and missing evidence, that
activation can be zero in all three cases. The exact atom state, evidence
identity, profile meet, and active mask resolve the collision. A normalized
vector of named surpluses may feed a declared downstream classifier or plan,
but that use defines a consumer whose own scale and evaluation obligations
must be stated.
